About Me

As a full-stack developer, I'm happy working on both the frontend and backend side to solve problems and create a better user experience.

Whether it's designing and building a landing page for a construction company, or building a secure and efficient server for room rentals, being involved in every step of the development process lends me a unique perspective in solving various problems using software.

I specialize with the MERN stack (MongoDB, Express.js, ReactJS, Node.js), though I also use PostgreSQL for apps better suited with relational databases. Currently, I am learning Redux, Next.js, and TypeScript to build more scalable and robust applications.
